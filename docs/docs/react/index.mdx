# Introduction
**Apollo Orbit** is an **<Link to="https://www.apollographql.com/docs/react/">Apollo Client</Link>** modular state management abstraction for **<Link to="https://react.dev">React</Link>**.

**<Link to="https://www.apollographql.com/docs/react/">Apollo Client</Link>**  is much more than a simple GraphQL client, it offers a comprehensive caching API that leverages GraphQL's rich type information system (schema) to provide features like cache normalization, optimistic UI and more.

**Apollo Orbit** aims to bridge the gap between **Apollo Client** and traditional state management libraries like Redux.
